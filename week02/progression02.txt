method Object.keys() เอาชื่อ object มาเป็น array ซึ่งง่ายต่อการเปรียบเทียบค่าของ object
ไม่ว่าจะเป็นความยาวของ string หรือ string มีค่าเท่ากันไหม จาก variable Shallow Equality ที่มีมาให้ใน silde หนูได้เพิ่ม variable ไป 3 variable เพื่อเช็คว่า 
1. ถ้าความยาวของค่าน้อยกว่าเมื่อเปรียบเทียบมาจะได้ค่าเท่ากันไหม 
2. ถ้า case-sensitive ขนาดตัวอักษรต่างกัน eg. Java // java (พิมพ์เล็ก/พิมพ์ใหญ่) จะได้ค่าที่เท่ากันไหม 
3. ถ้าความยาวเท่ากันแต่ค่า string ไม่ตรงกันจะได้ค่าที่เท่ากันไหม
ผลลัพธ์ที่ได้คือ 
1. true 
    isbn = 123456789 เท่ากัน
    title = "JavaScript" เท่ากัน
2. false
    isbn ต่างกัน 123456789 vs 12345
    title เท่ากัน แต่ แค่หนึ่ง property ต่างกันจะเป็น false ทันที
3. false
    isbn เท่ากัน
    title ต่างกันเพราะ function shallowEquality เช็ค case-sensitive eg. Java // java จะเป็น false ทันที ขนาดตัวอักษรต้องเท่ากัน Java === Java 
4. false 
    isbn เท่ากัน
    title ต่างกันเพราะ ถึงแม้ความยาวของ string เท่ากันมีอักษรพิมพ์เล็กพิมพ์ใหญ่เหมือนกัน (case-sensitive) แต่ถ้าค่าไม่ตรงกัน eg. Java // Test จะเป็น false ทันที
shallowEquality คือการเช็คค่าของ property ของ object ซึ่งการใช้ method Object.keys ทำเป็น array ทำให้สะดวกมากขึ้นในการใช้ for of วนลูปเช็คค่า
