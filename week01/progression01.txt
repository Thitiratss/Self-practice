basicarray01.js
เนื่องจากช่วงปิดเทอมไม่ได้เขียน code javascript เลยทำให้ลืมในสิ่งที่เรียนไปแล้วเยอะมาก ๆ ค่ะ
ในส่วน basicarray01.js เลยจะเป็นแค่การทบทวนสิ่งที่เคยเรียนหรือเคยเจอค่ะเพราะยังจำไม่ค่อยได้ค่ะ
1. ทบทวน spread operator และการเติมข้างหน้าข้างหลังค่ะ 
2. ลองใช้ for loop ควบคู่กับ array เพื่อหาค่ามาก
3. ข้อนี้ค่อนข้างยากเลยได้ใช้ AI ช่วยในบรรทัด cloneAr3[cloneAr3.length - 1 -i]; ค่ะ ตอนแรกแค่ -1 แล้วเลขออกมาเป็น 8 ทั้งหมดเลย พอถาม AI ถึงได้รู้ว่าต้อง -i ด้วย index จะได้เลื่อนตำแหน่ง array ไปเรื่อย ๆ ค่ะ
4. ทบทวนเรื่อง map ลองเขียนจากที่เพื่อนเคยตอบ solution ในห้องค่ะ ให้ map สร้าง array ใหม่ให้
5. ดึงตัวหน้าสุดและตัวสุดท้ายของ index แต่ใช้ spread ในการกระจาย string ค่ะ
6. แปลง string เป็น array โดยใช้ spread และเพิ่มข้อความข้างหน้าและข้างหลังค่ะ
7. ลองลบ index ออกและนำจำนวน empty index ค่ะ
8. ลองเพิ่ม array ใน spread และ ลองแบบเพิ่ม count ไปทีละ 1 เรื่อย ๆ จนจบที่ 100 ตามโจทย์ค่ะ
9. ลองรวม array ที่ซ้อนกันหลาย ๆ ชั้นให้อยู่ใน array เดียวค่ะ
