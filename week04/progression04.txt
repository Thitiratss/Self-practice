week04.js self-practice ที่หนูได้ทำมา หนูขอให้ AI ช่วยคิดโจทย์เกี่ยวกับ array method ที่เพิ่มความยากขึ้นมาอีกนิดหน่อยค่ะ
ข้อ 1. หานักเรียนที่ได้คะแนน >= 50 และอยู่ class "A"
หนูได้ลองใช้ .filter() กรองคะแนน และกรอง class พร้อม .map() เพื่อเอาชื่อออกมาค่ะ

ข้อ 2. หาคำที่ยาวที่สุด
หนูใช้ .reduce() ให้เปรียบเทียบความยาวของแต่ละคำค่ะ

ข้อ 3. รวมตัวอักษรไม่ให้ซ้ำ
หนูใช้ reduce กับ for of เพื่อวนตัวอักษรทีละตัวใน array และเช็ค includes เพื่อตรวจว่าซ้ำไหมตัวอักษรนี้เคยอยู่ใน result หรือยัง ถ้าไม่มีให้เพิ่มเข้าใน acc ถ้ามีให้ข้าม ไม่ให้เพิ่ม เพื่อไม่เอาตัวอักษรซ้ำ และใช้ join เพื่อให้ result กลายเป็น string ค่ะ

ข้อ 4. array ตัวเลข > 4, เรียงจากมากไปน้อย, แล้วบวก 1 ทุกตัว
หนูใช้ filter เพื่อกรองตัวเลขที่มากกว่า 4 ใช้ sort เพื่อเรียงจากมากไปน้อย และจากนั้น map ทุกค่าให้ + 1 เพื่อเพิ่มค่า 1 ให้ตัวเลชที่ผ่าน filter ค่ะ 

ข้อ 5. ผลรวมเลขที่หาร 5 ลงตัว แต่ต้องน้อยกว่า 40
หนูใช้ filter เพื่อกรองตัวเลขที่น้อยกว่า 40 ก่อน เพื่อให้เหลือเลขที่เราต้องการจริง ๆ จากนั้นให้ filter กรองเลขที่ หาร 5 ลงตัว แล้วใช้ reduce เพื่อ รวมผลลัพธ์ทุกตัวเลขค่ะ 

ข้อ 6. ตรวจสอบว่าทุกคำมีความยาว >= 3 หรือไม่ 
หนูใช้ map เพื่อให้ map คำใน array แปลงเป็น length ของคำ และหนูได้ใช้ every เพราะ every return ค่าเป็น boolean (true/false) เช็คค่าของ length ทุกคำใน array ว่า length >= 3 หรือไม่ แต่เพราะทุกค่ามี length >= 3 node ออกมาเลยได้ true ค่ะ

ข้อ 7. จัดกลุ่มนักเรียนตามห้อง A B C
ข้อนี้หนูใช้ reduce เพื่อวนทุก element ของ array และตรวจสอบว่ามีห้องนั้นอยู่หรือยัง (cur.class) ถ้ายังให้สร้างใหม่ ถ้ามีแล้วให้ push cur.name เข้าไปใน acc
